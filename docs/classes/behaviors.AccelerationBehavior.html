Class AccelerationBehavior

A Movement behavior that handles movement by applying a constant acceleration to all particles.

Example configuration:

{
    "type": "moveAcceleration",
    "config": {
         "accel": {
              "x": 0,
              "y": 2000
         },
         "minStart": 600,
         "maxStart": 600,
         "rotate": true
    }
}
