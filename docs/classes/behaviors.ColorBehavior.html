Class ColorBehavior

A Color behavior that applies an interpolated or stepped list of values to the particle's tint property.

Example config:

{
    type: 'color',
    config: {
        color: {
             list: [{value: '#ff0000' time: 0}, {value: '#00ff00', time: 0.5}, {value: '#0000ff', time: 1}]
        },
    }
}
