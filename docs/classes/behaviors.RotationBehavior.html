Class RotationBehavior

A Rotation behavior that handles starting rotation, rotation speed, and rotational acceleration.

Example configuration:

{
    "type": "rotation",
    "config": {
         "minStart": 0,
         "maxStart": 180,
         "minSpeed": 30,
         "maxSpeed": 45,
         "accel": 20
    }
}
