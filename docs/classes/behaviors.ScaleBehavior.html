Class ScaleBehavior

A Scale behavior that applies an interpolated or stepped list of values to the particle's x & y scale.

Example config:

{
    type: 'scale',
    config: {
         scale: {
             list: [{value: 0, time: 0}, {value: 1, time: 0.25}, {value: 0, time: 1}],
             isStepped: true
         },
         minMult: 0.5
    }
}
